  1. Mengapa setiap bahasa pemrograman itu terdapat framework, apa kegunaan framework ?
    Framework adalah kerangka kerja yang dibuat berdasarkan dari bahasa pemograman itu sendiri. Menurut saya setiap bahasa pemograman terdapat framework karena terdapat fungsi/metode yang sudah dikumpulkan, yang umumnya sering digunakan untuk membuat sebuah aplikasi.
    kegunaan Framework:
    1. Mempercepat penulisan dan pembuatan aplikasi
    2. Memudahkan penggunaan metode/fungsi
    3. Mengatur aplikasi agar sesuai dengan konsep MVC dan OOP
  2. Jelaskan mengenai fungsi dan penggunaan if else, while, while do, switch case, for, foreach?
    fungsi dari :
    1. if else : Melakukan pengecekan sebuah kondisi, yang mana jika kondisi tepenuhi, maka kode di dalam if akan dijalankan, namun jika kondisi tidak terpenuhi, maka kode else akan dijalankan
    2. while : Menjalankan sebuah perulangan jika kondisi tepenuhi, kode baru dijalankan
    3. while do : Untuk menjalankan sebuah peulangan, namun akan dijalankan kodenya sekali sebelum di cek kondisi pada whilenye
    4. switch case : Untuk melakukan pengecekan pada kondisi tertentu yang sudah ditetapkan, maka kode akan dijalankan sesuai kondisi
    5. for : Untuk melakukan perulangan, dimulai dari angka pertama yang ditetapkan, menuju angka terakhi, baik melakukan penambahan atau pengurangan
    6. foreach : Untuk melakukan perulangan, selama suatu nilai berada dalam rentang di dalam foreach
  3. Apa itu constructor, method, class, object, variabel, parameter, function, procedure ?
    1. Constructor : Fungsi yang dijalankan ketika class pertama kali di inisialisasi
    2. Method : Kumpulan kode yang dibungkus dalam satu fungsi, untuk dilakukan pemanggilan dari sebuah class, atau objek class
    3. Class : Kumpulan method, variabel, constructor dan function yang digunakan untuk membuat sebuah objek
    4. Object : Instance dari class yang sudah di inisialisasi
    5. Variabel : Digunakan untuk menyimpan sebuah nilai yang dapat berubah
    6. Parameter : Variabel yang digunakan untuk digunakan pada function atau method, yang mana nilainya berasal dari luar function/method
    7. Function : blok kode yang dapat digunakan berulang kali pada sebuah program, dan terdapat return value
    8. Procedure : blok kode yang dapat digunakan berulang kali pada sebuah program, namun tidak terdapat return value

    HMVC - adalah struktur lanjutan dari MVC, dimana controller, view dan model dibuat lebih terperinci
